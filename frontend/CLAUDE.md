# Frontend Guidelines (Phase III)

## Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **HTTP Client:** Axios

## Features

### Todo Management
- Lists tasks from the backend.
- Adds new tasks.
- Updates/Deletes tasks.
- **Component:** `src/components/TodoApp.tsx`

### AI Chatbot
- Floating chat interface (bottom-right).
- Connected to `POST /api/chat`.
- Supports natural language task management.
- **Component:** `src/components/ChatInterface.tsx`

## Commands
- **Install:** `npm install`
- **Run Dev:** `npm run dev`
